# Architecture
- /backend: Node.js with TypeScript and Express. Database ORM is Prisma.
- /frontend: Next.js with TypeScript and Tailwind CSS.
- /shared: Contains shared TypeScript interfaces.

# Rules for AI Agents
- Strictly adhere to your assigned directory. Do not modify root configs.
- Always use strong typing. Avoid `any`.
- Create new files rather than editing existing central routers.
